Jupyter notebooks consist of cells that can include either writing or code. Let's explore how writing code in Jupyter works.

(a)

To write and execute code, your notebook needs to have a kernel, that is, an associated programming language. Select a Python 3 kernel from the notebook menu. (If there are multiple versions of Python 3 available, it doesn't matter which one you choose. Do not choose a Python 2 kernel, however.)

Once you've done this, edit the contents of the below cell to read import sys; sys.version and hit Shift+Enter.

(b)

Most likely, the result of entering the code import sys; sys.version into the above cell just make the text “import sys; sys.version” appear. This is because the editable cells in this notebook are set to “Markdown” mode by default, which is used for writing text, such as the paragraph you're currently reading. (If the above cell wasn't already in Markdown mode, use the menu to switch it to Markdown mode, so the output is simply “import sys; sys.version”.)

Now enter import sys; sys.version into the below cell, and use the menu to switch the cell type to “Code”. Now using Shift+Enter should display something like '3.6.9 (default, Apr 18 2020, 01:56:04) \n[GCC 8.4.0]', the version of the Python kernel you selected.
