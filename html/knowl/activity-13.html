Activity 2.2.3.

Some LaTeX notation is rather straight-forward, with literal notation such as + and -, and intuitive notation such as ^ for superscripts and _ for subscripts: a_0+a_1^2=-a_2^3 produces \(a_0+a_1^2=-a_2^3\text{.}\)

But multiplication cannot be represented by, say, x, since that represents the variable \(x\text{.}\) (Instead we write 2 \times 3 to produce \(2 \times 3\text{.}\))

Even professionals usually know just a small subset of all the LaTeX commands available. Instead, references like Dr. Burn's TeX Command Reference and tools like DeTeXify allow for easy lookup of LaTeX commands as needed.

For each of the following samples of mathematical notation, don't peek at their source markup. Instead, use the above resources to replicate the notation as best as possible in a Markdown cell. After you've done as well as possible on your own, then compare with the corresponding source markup.

(a)
\begin{equation*} \det(A) = x_{11}x_{22}-x_{12}x_{21} \end{equation*}
(b)
\begin{equation*} \sin(\alpha+\beta) = \sin(\alpha)\cos(\beta)+\cos(\alpha)\sin(\beta) \end{equation*}
(c)
\begin{equation*} \sigma(t) = \langle 0,t_1,2t_2,\dots,nt_n \rangle \end{equation*}
(d)
\begin{equation*} \overline{x} = \frac{1}{n}\left(\sum_{0\leq i<n}x_i\right) \end{equation*}
