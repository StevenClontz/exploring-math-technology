Throughout this course, you will complete the activities from this text within Jupyter notebooks hosted on CoCalc.com, some individually, and some within groups. Your first activity is to set these Jupyter notebooks up for yourself!

  1. First, either create an account at CoCalc.com, or log in with the account provided by your instructor.

  2. Go to TODO https://share.cocalc.com/share/d17525…, and use the “Open in CoCalc” button to create a new project with your notebooks for the course.

  3. Open the Settings for this project, and add your instructor's account as a Collaborator. This allows them to see your work, and unlock the premium features required for this course (such as importing data from other websites to analyze).

  4. Finally, navigate to the notebook associated with this activity. (You may need to select a “kernel” to open it; that choice is not important for now, so pick anything.) In the editable textbox appearing below this activity in the notebook, double-click to edit, erase any default content, and type the well-worn phrase of computing: Hello World!. Tap Shift+Enter to save your note.
