Since ultimately data is stored as bits, anything handled by a computer needs to be represented somehow as a (binary) number. For example, a character is a letter or other glyph (such as an emoji üòÉ) that has a unique representation as a counting number.

To denote a character in Python or SageMath, surround it with either single quotes 'x' or double quotes "x". Then the code ord('x') (or ord("x")) will reveal the number that is used to represent it in memory (for x, that number is \(120\)), called a Unicode value.
