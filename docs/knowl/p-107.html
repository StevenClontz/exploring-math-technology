For example, suppose we now want to assume \(x=6,y=0,z=-3\) and compute the corresponding value for the expression \(x^2+4yz\) saved as my_expression. You can check that \(6^2+4(0)(-3)=36\text{,}\) either by hand or evaluating 6^2+4*0*(-3). But to have SageMath compute this for you immediately, you could use the following syntax: my_expression(x=6,y=0,z=-3).
