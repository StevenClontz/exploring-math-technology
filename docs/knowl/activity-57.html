Activity 8.1.3.

SageMath provides the points() tool to plot points in the Cartesian plane, demonstrated in 8.1.2.

(a)

Modify and run 8.1.2 in a Code cell to plot the following complex numbers.

  • \(\displaystyle 3-5i\)
  • \(\displaystyle 4i\)
  • \(\displaystyle -2\)
  • \(\displaystyle -4+3i\)
(b)

Modify and run 8.1.2 in a Code cell to plot the following complex numbers. (You'll need to figure out how to resize the visible portion of the plane.)

  • \(\displaystyle 10+i\)
  • \(\displaystyle 9\)
  • \(\displaystyle -4+8i\)
