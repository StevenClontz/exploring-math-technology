Task 4.1.2.d.

Visit the website https://vincentarelbundock.github.io/Rdatasets/datasets.html to get a CSV with the transactions for a (fictional) pizza parlor for the year 2015. (Hint: Use Ctrl+F to search for “pizza”.) Rather than downloading it, right-click and copy the URL to the CSV file.

Import this data directly by entering Table.read_table("https://example.com/fix/this/url/to/point/to/your/data.csv") into a Code cell, replacing the string with the URL you copied. (Note that while there are almost fifty thousand transactions in this dataset, only a few are displayed by default. 4.1.6 gives the first three rows.)
