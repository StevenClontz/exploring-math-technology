Task 1.3.3.a.

To assign a value to a CS variable, the assignment operator = is used. It's considered best practice to give CS variables semantic, that is, descriptive names such as temperature_in_room, using lowercase letters and underscore _ characters.

For each member of your group, use a code cell to assign a CS variable storing the number of pens (or pencils, or any object you'd like) each person has with them. An example is given in 1.3.2; note that only a single Code cell is necessary, but each assignment should be made on a separate line.
