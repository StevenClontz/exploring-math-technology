Data is often stored in a spreadsheet or database as a table. An illustration of this is given in 4.1.4.

Tables can be created programmatically by using the technique from 4.1.1 or by modifying 4.1.5.

Run from datascience import Table in a Code cell, and then copy 4.1.5 to another Code cell. Finally, append the student data you created in the previous task.

If importing Table fails, run the code from 4.1.1 to make the datascience library available in your project.
