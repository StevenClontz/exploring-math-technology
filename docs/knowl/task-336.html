Task 10.1.3.b.

Any pair of vectors forms a basis, provided they don't point in exactly the same (or opposite) directions.

Modify 10.1.2 to set basic1 and basic2 equal to the non-standard pair of basis vectors \(\vec b_1=(1,2)\) and \(\vec b_2=(-1,1)\text{,}\) and then visualize how the vectors \(-\vec b_1+3\vec b_2\text{,}\) \(3\vec b_1+4\vec b_2\text{,}\) and \(-2\vec b_1-5\vec b_2\) appear in this modified coordinate system.
